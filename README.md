Install script for Hyper_ client and hypercontainer
===================================================

# Usage

## install hypercli for Hyper_

website: https://hyper.sh

```
$ curl -sSL https://hyper.sh/install | bash
or
$ wget -qO- https://hyper.sh/install | bash
```
